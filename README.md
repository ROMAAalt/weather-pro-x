# 🌦️ Weather Pro X

A modern weather forecast web application that provides real-time weather updates, interactive maps, weather visualization, and location-based forecasts using OpenWeatherMap API, Chart.js, and Leaflet.js.

## ✨ Features

### 🌍 Weather Data

* Real-time weather updates using OpenWeatherMap API
* Search weather for any city worldwide
* Temperature, humidity, wind speed, pressure, and visibility data

### 📅 Forecast System

* Hourly weather forecast
* 5-day weather forecast

### 🗺️ Interactive Maps

* Interactive map powered by Leaflet.js
* Dynamic city location markers

### 📊 Data Visualization

* Temperature trend charts using Chart.js

### ⚡ Additional Features

* Geolocation-based weather detection
* Favorites system
* Search history tracking
* Dark mode support
* LocalStorage persistence
* API response caching

## ⚙️ Installation & Setup

```bash
# Clone repository
git clone https://github.com/ROMAAalt/weather-pro-x.git

# Move into project folder
cd weather-pro-x

# Open index.html in your browser
```

For the best development experience, use the Live Server extension in VS Code.

---

## 🔑 API Configuration

Get your free API key from:

https://openweathermap.org/api

Replace the API key in your JavaScript file:

```javascript
const API_KEY = "YOUR_API_KEY";
```

---

## 🌐 Deployment

This project can be deployed using GitHub Pages:

1. Push the project to GitHub.
2. Open Repository Settings.
3. Navigate to Pages.
4. Select:

   * Source: Deploy from branch
   * Branch: main
5. Save changes.

Your website will be available at:

https://ROMAAalt.github.io/weather-pro-x/
