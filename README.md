# 🌦️ Weather Pro X

A modern weather forecast web application that provides real-time weather updates, interactive maps, and data visualization using OpenWeatherMap API, Chart.js, and Leaflet.js.

## ✨ Features

### 🌍 Weather Data
- Real-time weather updates using OpenWeatherMap API
- Search weather for any city worldwide
- Temperature, humidity, wind speed, pressure, visibility

### 📅 Forecast System
- Hourly weather forecast
- 5-day forecast view

### 🗺️ Map Integration
- Interactive map using Leaflet.js
- Live location marker based on city

### 📊 Visualization
- Temperature trend chart using Chart.js

### ⚡ Extra Features
- Auto-complete city search
- Geolocation-based weather detection
- Favorites system
- Search history tracking
- Dark mode support
- LocalStorage persistence
- API caching
